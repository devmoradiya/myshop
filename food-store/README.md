# Food Store

A simple static site inspired by Swiggy to browse restaurants with search, filters, and sorting.

## Preview locally

From the workspace root:

```bash
python3 -m http.server 8080 -d /workspace/food-store
```

Then open `http://localhost:8080` and navigate to `food-store/` if needed.

## Structure

- `index.html` — markup and layout
- `styles.css` — responsive styles
- `app.js` — data, rendering, interactions
- `assets/` — favicon and future assets
